The goal of this project was to repurpose the e-waste from my old laptop and gain hands-on experience with virtualization, network administration, system administration, cybersecurity, firewalls, access control, Docker containers, and much more.

I used Proxmox as the hypervisor to host my virtual machines and containers. In my configuration, I have:

  • Pi-Hole container
  • Kali Linux machine
  • Ubuntu Server machine
  • CasaOs on Debian machine
  • OpenMediaVault machine
  • Debian machine
  • WUbuntu machine

This server allows me to host my personal website on the Ubuntu Server VM, set up a NAS using Samba share, and connect to it from anywhere in the world, thanks to Tailscale.
