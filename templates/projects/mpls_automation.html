The goal of this project was to reperpous the e-waste of my old laptop and gain hands-on experience on virtualisation, network administration, system administration, cybersecurit, firwalls, accesses control, docker containers and much more.

I used Proxmox as the hypervisor to host my virtual machines and containers. In my configuration, I have:

  • Pi-Hole container
  • Kali Linux machine
  • Ubuntu Server machine
  • CasaOs on Debian machine
  • OpenMediaVault machine
  • Debian machine
  • WUbuntu machine

This server allows me to host my personal website on the Ubuntu Server VM, as well as setting up a NAS using samba share and conncting to it from everywhere in the world thanks to Tailscale.
