Network Configuration Automation

MPLS is widely used in telecommunications and enterprise networks due to its ability to improve data transfer speeds and enhance routing efficiency. However, the manual configuration required for MPLS routing can be complicated and time consuming, particularly when working with extensive network topologies. This project explores a solution: using Python scripting to automate the MPLS network configuration within GNS3, streamlining deployment and ensuring consistency.
