This was our a school project developed in a team of 5 students; Kaoutar El Arrag, yaazza ayoub, ESSGHEIR Kawtar, Amina SAKOUTI and myself . The objective of this project was to design an IoT device to assist users in plant care. The device monitors the plant's environment twice daily, providing users with insights into their plants' health. Additionally, it sends alert notifications if any intervention is required to improve the plants' conditions.
The materials used in this project were:

  • ESP8266 Development board
  • LM393 Soil moisture sensor
  • GL5516 LDR
  • DHT11 Ambient humidity and temperature sensor

I was responsible for assembling the electronics, designing the PCB, and programming the ESP8266. The board was configured to initiate an interface that enables users to connect to their Wi-Fi network, read sensor data, and transmit it to a server via an API I developed using Pythonâ€™s FastAPI. Additionally, I designed a 3D model of the device using Fusion 360.
